Tornis.

Tornis is a minimal JavaScript library that watches
the state of your browser's viewport,
allowing you to respond whenever something changes.

Think of it as data store for your viewport, giving you
access to screen size, mouse cursor position,
scroll position,gyroscope and more

Get Tornis
Rekishi.

Rekishi is minimal wrapper for the History API that provides
additional pub/sub functionality

If you need to create dynamic transitions between different pages or
different types of content, then Rekishi can help.
